# 주제
반려동물 동반 애견 호텔 검색 웹 사이트

## 기능
![기능](/uploads/f712d71bf35fea267d4b03509f01a7a8/기능.png)

- 펜션과 호텔 검색 - 위치, 날짜, 인구별로 검색이 가능합니다. 가격별로 정렬이 됩니다.
- 지도 - 숙소 주변 근처의 동물병원, 공원을 보여줍니다.
- 회원 관리 -  SNS 소셜 로그인, 가입하기
- 공지사항 관리
